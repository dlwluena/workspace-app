# Workspace App (Second Brain)

> A minimalist, all-in-one personal organization tool designed to function as your "Second Brain". Manage projects, tasks, and daily journals in one distraction-free interface.

## Features

### 1. Inbox (Rapid Capture)
* Quickly dump ideas, tasks, and reminders without worrying about organization.
* Simple checklist interface.

### 2. Workspace (Project Management)
* **Custom Sections:** Create folders for different areas of your life (e.g., WORK, STUDY, HOBBIES).
* **Pages:** Add unlimited pages within sections.
* **Task Tracking:**
    * Set **Priority** (Low, Medium, High).
    * Track **Status** (To Do, In Progress, Done).
    * Table view for detailed management.

### 3. Daily Journal (With History)
* A dedicated space for daily logs and thoughts.
* **Date Picker:** Navigate to any specific date.
* **Save & Reset:** One-click button to save the current day's entry to history and reset the view for a fresh start.
* **Journal History:** Access past entries via the sidebar and delete them if needed.

### 4. Privacy & Data
* **Local Storage:** All data is stored locally in your browser's `localStorage`.
* **No Server:** Your data never leaves your device.
* **Persistent:** Your data remains safe even if you refresh or close the browser.

## How to Run Locally

1. **Clone the repository**
   ```bash
   git clone [https://github.com/dlwluena/workspace-app.git](https://github.com/dlwluena/workspace-app.git)
   ```
2. Open the project Navigate to the project folder and open index.html in your browser.
  ```bash
    cd workspace-app
    # On Mac:
    open index.html
    # On Windows:
    start index.html
```
